# security-monitor
#### 介绍
这是一个linux驱动开发的安防监控项目，硬件环境为Cortex-A9开发板和zigbee等。通过传感器进行实时采集环境数据，例如温度与湿度。再通过zigbee把采集到的环境信息发送给服务器，由A9的采集数据线程接受环境参数。通过共享内存与信号量将环境信息上传网页。并通过消息队列接收客户端请求进而控制硬件。同时结合摄像头实现图形远程显示，保证客户能远程掌握
监控环境的实时情况。


#### 软件架构
开发平台： Ubuntu14.04  vs2012 vs2017

开发环境： linux3.14.0、uboot2013.01、gcc4.6.4

通信协议： 串口、ZigBee、http协议等

数据库 ： sqlite3

#### 硬件架构

主机系统支持：PC(LINUX)等主流主机。

存储设备：  SD卡、EMMC、SCSI或IDE接口的硬盘。

终端设备：  ZigBee、STM32、RFID、cortex-A9、PC显示屏。

#### 使用说明

1.  前期准备为一些项目需求以及结构体设计的文件
2.  securityMonitor为移植好的boa、驱动等源码，只需要进去对应目录make生成执行文件，make install将文件拷贝到cortex-A9的根文件系统
3.  rootfs为配置好的根文件系统，设备树和内核为配置好的设备树和内核
